#  Resume Builder - MERN Stack Application

A full-stack Resume Builder Web Application built using the MERN stack (MongoDB, Express.js, React.js, Node.js).

This application allows users to create, edit, manage, and download professional resumes with secure authentication.

---

##  Tech Stack

###  Frontend
- React.js
- Axios
- CSS / Tailwind CSS

###  Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication

##  Features

-  User Registration & Login
-  JWT Based Authentication
-  Create Resume
-  Edit Resume
-  Delete Resume
-  Download Resume
-  Responsive UI
-  RESTful API Integration
